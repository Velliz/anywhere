Konfigurasi Dasar

Untuk membuat sebuah kode QR. Anda hanya perlu mengirim data dalam bentuk teks ke API URL yang sudah disediakan.

API URL
{URL}qr/render?data=[data]

Contoh
{URL}qr/render?data=admin@example.co.id

QR Code Options

Options Value Example
output jpeg|jpg|png|svg http://localhost/anywhere/qr/render?data=05b810384abc26a2365e1108501534a2?output=jpeg
size 1 to 100 http://localhost/anywhere/qr/render?data=05b810384abc26a2365e1108501534a2?size=9
margin 1 to 100 http://localhost/anywhere/qr/render?data=05b810384abc26a2365e1108501534a2?margin=15
