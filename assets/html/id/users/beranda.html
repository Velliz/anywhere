Pembuat PDF

Nama template Ukuran kertas Orientasi Aksi

Catatan PDF

HTML tidak perlu menyertakan tag `head` atau `body`. Gunakan elemen konten seperti `div` atau `table` dan pastikan semua tag ditutup dengan benar.

Pastikan CSS valid saat memakai variabel JSON. Hindari deklarasi kosong seperti `color: ;` atau `background: ;`. Gunakan nilai default misalnya `color: #000000; background: #0067AA;`.

Proses rendering menggunakan tag PTE {!}. Dokumentasi tersedia di https://github.com/Velliz/pte
